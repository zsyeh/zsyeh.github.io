## 命令行
```bash
export http_proxy=http://<windows-host-ip>:<port>
export https_proxy=http://<windows-host-ip>:<port>
```
## git
```bash
git config --global http.proxy 'http://127.0.0.1:http端口号'
git config --global https.proxy 'https://127.0.0.1:https端口号'
```
